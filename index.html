Overview

With Advaita 2019 just round the corner, gear up for a fun challenge with your partner!
Presenting the "Web Dev challenge 2019" for Advaita.

Event Details:

Team Size: 1-2
Event Duration: 4th February 2019 18:00 - 5th February 2019 18:00 (24 hrs)

Topic

Expenses Tracker

Features expected:

  1. User can signup and thereafter login
  2. Store his expenses.
  3. Tag expenses by categories.
  4. Display expenses.
  5. Show expense statistics for last month, last year and etc using graphs (optional)

NOTE:
You are free to use any Language / Tech Stack.

    100 points

  • Features

    100 x 5 = 500 points
