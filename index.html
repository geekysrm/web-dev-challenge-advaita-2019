Topic

Online Auction Platform

Features expected:

  1. User can signup and thereafter login.
  2. User can host an auction by providing details of the product like description, photos, base bid price, start and end time etc.
  3. Provide a dashboard to display all live auctions.
  4. Other users can place bid on live auctions.
  5. Once the auction is over the product will sold to the highest bidder.

NOTE:
You are free to use any Language / Tech Stack.
Extra features will get bonus points.
